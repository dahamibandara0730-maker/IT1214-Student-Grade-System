&#x20;IT1214 Student Grade Management System



&#x20;Project Purpose



&#x20;The system helps teachers store, manage, and analyze student academic records digitally instead of using manual paper-based methods.







&#x20;Features

\- Add new student records (ID, Name, Marks)

\- Display all student records

\- Search student by Student ID

\- Calculate average marks of all students

\- Simple menu system







&#x20;Technologies Used

\- Java

\- Object-Oriented Programming 

\- ArrayList

\- Git \& GitHub (Version Control)







&#x20;How to Run



&#x20;javac Student.java Main.java



&#x20;java Main
